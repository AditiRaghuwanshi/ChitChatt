ChitChat – Real-Time Chat Application

ChitChat is a full-stack real-time chat application that enables seamless communication through instant messaging, group chats, and live updates. It is built with a scalable architecture using modern web technologies.

Features
Real-time messaging using Socket.IO
One-to-one and group chat functionality
Typing indicators and online/offline user status
Message persistence with database storage
Responsive user interface across devices
REST API integration for user and chat management
Tech Stack
Frontend
React
Redux Toolkit
CSS / Tailwind CSS
Backend
Node.js
Express.js
Database
MongoDB
Real-Time Communication
Socket.IO
