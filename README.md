# ChitChat – Real-Time Chat Application

ChitChat is a full-stack real-time chat application enabling seamless communication.

## Features

- Real-time messaging using Socket.IO
- One-to-one chat functionality
- Group chat support with multiple users
- Typing indicators for active conversations
- Online and offline user status tracking
- Message persistence using database storage
- Responsive UI across mobile, tablet, and desktop
- REST API integration for user authentication and chat management

## Tech Stack

### Frontend
- React
- Redux Toolkit
- CSS / Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Real-Time Communication
- Socket.IO
